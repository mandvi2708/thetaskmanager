This project is a full-stack web application designed to simulate real-world collaboration tools like task boards and project managers.

Users can:

Create and manage projects
Assign tasks to team members
Track task progress visually
Monitor productivity through dashboards
✨ Features
🔐 Authentication
User Signup & Login
Secure authentication using Supabase Auth
Persistent user sessions
📁 Project Management
Create new projects
Add/remove members
Role-based access (Admin / Member)
✅ Task Management
Create tasks with:
Title
Description
Due Date
Priority
Assign tasks to users
Update task status:
To Do
In Progress
Done

📊 Dashboard
Total tasks overview
Tasks by status
Overdue tasks
Tasks assigned per user
🛡️ Role-Based Access
Admin:
Manage members
Create/Delete tasks
Member:
View assigned tasks
Update task status
⚡ Real-Time Updates
Instant UI updates using Supabase subscriptions
🛠️ Tech Stack
Frontend
Next.js (App Router)
Tailwind CSS
shadcn/ui
Backend
Supabase (Auth + Database + Realtime)
Database
PostgreSQL
